# Gemma Query Wikipedia Plus

## Description
Advanced Wikipedia search and summarization tool with multilingual support. Features fuzzy search, article summaries, infobox extraction, and clean UI for exploring knowledge.

## Features
- Fuzzy search for accurate article matching
- Multilingual support (20+ languages)
- Article summaries and extracts
- Infobox extraction for key facts
- Clean, responsive UI
- Mobile-optimized interface

## Usage
1. Enter search query in any supported language
2. Select target language for results
3. View summary and key information
4. Explore related articles

## Supported Languages
English (en), Spanish (es), French (fr), German (de), Italian (it), Portuguese (pt), Russian (ru), Japanese (ja), Korean (ko), Chinese (zh), Arabic (ar), Hindi (hi), and more.

## Technical Details
- Uses Wikipedia API for data retrieval
- Fuzzy search for better matching
- Extracts structured data from infoboxes
- Responsive design for all devices
