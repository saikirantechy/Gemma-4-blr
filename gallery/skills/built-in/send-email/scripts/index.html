📧 Gemma Send Email

Native Android intent skill for composing and sending emails via the AI Edge Gallery app

About This Skill

The Send Email skill is a Native Intent skill. Instead of running JavaScript in a WebView, it uses the Android run_intent tool to launch the device's email client directly. The LLM (Gemma) extracts the recipient, subject, and body from your natural language request and passes them to the native email composer.

How it works:
🗣️ User Prompt → 🤖 Gemma parses intent → ⚡ run_intent tool → 📱 Native email app
Example prompts you can use in the app:
  • ✉️ "Send an email to john@example.com about the project update"
  • ✉️ "Email Sarah about the meeting tomorrow: We need to finalize the agenda"
  • ✉️ "Draft an email to my team: Subject: Sprint Review — Body: Please review the latest changes"

⚙️ Technical Details

The skill uses the Android Intent system via the run_intent tool. The LLM calls this tool with two parameters:

Parameter Description Example
intent The native action to perform send_email
parameters JSON string with intent-specific fields { "extra_email": "...", ... }

Email-specific fields in parameters:

Field Type Description
extra_email String The recipient email address
extra_subject String The email subject line
extra_text String The email body content

📖 Developers: To add new native intents, see IntentHandler.kt in the AI Edge Gallery Android source.
