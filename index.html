Gemma Pocket
Assistant

An on-device AI utility assistant powered by Gemma + LiteRT-LM. Wikipedia intelligence & QR generation — fully offline, privacy-first, running on your Android device.

⭐ View on GitHub 📱 Install Skill
✨ What it does
🔎

Wikipedia Search

Ask about any person, place, or event. Gemma queries Wikipedia and summarizes the result locally — no cloud inference needed.

📱

QR Code Generation

Turn any URL or text into a scannable QR code instantly. Auto-prefixes https:// and renders the image right in your chat.

⚡

Offline-First AI

Inference runs entirely on-device via LiteRT-LM. No latency from cloud round-trips. Works on any Android with AI Edge Gallery.

🔒

Privacy-Focused

Your prompts never leave your device. The Gemma model handles all reasoning locally — only tool calls fetch external data.

⚙️ How It Works
1

User sends a prompt

You type naturally — "Who is Sundar Pichai?" or "Generate QR for sktnexus.com"

2

Gemma reads SKILL.md and routes

The on-device model detects the intent and calls the correct tool: wikipedia.html or qrcode.html

3

JS skill executes in hidden WebView

Wikipedia: fuzzy search + infobox parse via REST API. QR: qrcodejs CDN renders to canvas → base64 PNG.

4

Gemma summarizes and responds

The result is passed back to the model, which delivers a clean 2-3 sentence answer — or renders the QR image directly in chat.
